# Configuration Documentation

## Authentication Configuration

- **authBaseUrl**: Keycloak authentication base URL
  - Default: `https://auth.datenportal.localhost`
  
- **authClientId**: Keycloak client ID for the data portal
  - Default: `dataportal`
  
- **authRealm**: Keycloak realm name
  - Default: `dataportal`

## Backend Configuration

- **backendBaseUrl**: Backend API base URL
  - Default: `https://api.datenportal.localhost`

## Application Metadata

- **copyrightOwner**: Copyright owner name
  - Default: `FDPG+ Team`
  
- **copyrightYear**: Copyright year
  - Default: `2025`
  
- **email**: Contact email address
  - Default: `info@forschen-fuer-gesundheit.de`

## UI Configuration

- **stylesheet**: Stylesheet theme name
  - Default: `FDPGTheme`
  
- **version**: Application version
  - Default: `6.6.0`
