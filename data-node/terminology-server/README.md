## SNOMED CT

Please use the International Edition of SNOMED CT, even tought a Germany Edition exists. The correct KDS version can be found [here](https://github.com/medizininformatik-initiative/kerndatensatz-meta/wiki/Terminology-Version-Policy). Newer Versions can be used, because all older versions are available automatically.
